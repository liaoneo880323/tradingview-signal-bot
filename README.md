\# TradingView 信號機器人



將TradingView警報通過Webhook轉發到Telegram。



\## 部署步驟



1\. 在Vercel部署此項目

2\. 設置環境變量：

&nbsp;  - `TELEGRAM\_BOT\_TOKEN`

&nbsp;  - `TELEGRAM\_CHAT\_ID`

3\. 在TradingView中配置Webhook警報



\## Webhook URL

部署後你的Webhook URL為：

`https://你的應用.vercel.app/api/webhook`
